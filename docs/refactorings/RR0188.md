## Replace foreach with for and reverse loop

| Property           | Value                                     |
| ------------------ | ----------------------------------------- |
| Id                 | RR0188                                    |
| Title              | Replace foreach with for and reverse loop |
| Syntax             | foreach statement                         |
| Enabled by Default | \-                                        |

### Usage

#### Before

```csharp
foreach (object item in items)
{
    yield return item;
}
```

#### After

```csharp
for (int i = items.Count - 1; i >= 0; i--)
{
    yield return items[i];
}
```
